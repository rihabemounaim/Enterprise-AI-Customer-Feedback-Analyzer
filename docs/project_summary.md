# Project Summary

## Enterprise AI Customer Feedback Analyzer

This project demonstrates the design of an enterprise-ready Generative AI workflow using SAP Generative AI concepts.

The system automates customer email processing by:
- analyzing customer sentiment,
- extracting structured business information,
- detecting multiple issues,
- assigning priorities,
- and generating machine-readable JSON outputs.

The project integrates advanced Prompt Engineering techniques including:
- Prompt Hardening,
- Grounding,
- Few-Shot Prompting,
- and Retrieval-Augmented Generation (RAG).

Several Large Language Models (LLMs) were evaluated comparatively, including GPT-4o, Gemini, and Mistral AI, in order to determine the most reliable model for production deployment.

## Main Objectives
- Automate customer feedback analysis
- Improve business workflow efficiency
- Reduce manual processing time
- Enable structured AI-driven routing
- Enhance enterprise decision support
